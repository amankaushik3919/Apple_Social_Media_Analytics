# 🍎 Apple Social Media Analytics

> **Excel-Powered Business Insights**: Analyzing Apple's social media sentiment, engagement trends, and brand perception using lightweight, reproducible spreadsheet workflows.

---

## 🎯 Business Purpose

This repository contains an Excel-based analytics workbook to extract actionable business intelligence from Apple's social media presence — no coding required.

✅ Monitor brand sentiment around product launches (iPhone, Vision Pro, Services)  
✅ Track engagement performance across platforms (X/Twitter, Instagram, Reddit)  
✅ Identify emerging consumer themes: privacy, sustainability, AI features  
✅ Benchmark Apple's social performance vs. competitors  
✅ Generate executive-ready charts & summaries for stakeholder reporting

### 🔍 Workbook Tabs Overview
1. **Raw_Data**: Imported social mentions (text, date, platform, engagement metrics)  
2. **Sentiment_Score**: Manual/rules-based sentiment tagging (Positive/Neutral/Negative)  
3. **Pivot_Analysis**: Pre-built pivot tables for trend slicing (by product, region, time)  
4. **Charts_Dashboard**: Visual summaries: sentiment over time, top hashtags, engagement heatmaps  
5. **Insights_Log**: Space to document business takeaways & action items  

---

## 🛠️ How to Use

1. **Open** `Apple_Social_Analytics.xlsx` in Microsoft Excel (2019+) or Google Sheets  
2. **Refresh Data** (if connected to live sources):  
   - *Data* → *Refresh All* (for Power Query connections)  
   - Or manually paste new exports into `Raw_Data` tab  
3. **Explore**:  
   - Use slicers on the Dashboard tab to filter by product, date, or platform  
   - Review auto-updating charts for sentiment & engagement trends  
4. **Export**: Copy visuals or pivot summaries for reports/presentations  

> 💡 *No macros or VBA required — fully compatible with Excel for Web & mobile viewers.*

---

## 📥 Data Sources (Manual Export Workflow)

Since this is a lightweight Excel project, data is gathered via:
- Platform native exports: X Analytics, Instagram Insights, Reddit search exports  
- Third-party tools: Brand24, Hootsuite, or Talkwalker CSV exports  
- Manual curation: Copy-paste high-impact posts for qualitative review  

*Tip: Use Excel's **Power Query** (Data → Get Data) to semi-automate future imports.*
