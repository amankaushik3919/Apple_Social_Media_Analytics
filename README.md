# 🍎 Apple Marketing Data Analytics

> **Excel-Based Business Analysis**: Exploring Apple marketing performance, customer trends, and sales insights using practical spreadsheet analytics.

---

## 🎯 Purpose

This repository contains Excel workbooks for hands-on practice in marketing data analysis — focused on Apple-related datasets to build business intelligence skills.

✅ Clean, transform, and analyze marketing & sales data in Excel  
✅ Practice pivot tables, formulas, and visual storytelling  
✅ Explore customer segmentation, product performance, and regional trends  
✅ Build portfolio-ready analysis for business/analytics roles  

---

## 📁 Repository Files

| File | Description |
|------|-------------|
| `Apple Marketing Data.xlsx` | Apple-focused marketing/sales dataset for analysis practice |
| `Excel practice Sheet.xlsx` | Workbook with exercises: formulas, pivots, charts, and data cleaning |
| `US Superstore (1).xlsx` | Sample retail dataset for comparative analysis & skill building |
| `README.md` | This file |

---

## 🛠️ How to Use

1. **Open any `.xlsx` file** in Microsoft Excel (2019+) or Google Sheets  
2. **Explore the tabs**: Raw data, cleaned tables, pivot analyses, and visual dashboards  
3. **Practice key skills**:  
   - `VLOOKUP`/`XLOOKUP`, `IF` logic, conditional formatting  
   - PivotTables for segmentation (by product, region, date)  
   - Charts: trend lines, bar comparisons, heatmaps  
4. **Extend**: Add your own analysis tabs, save insights, or export visuals for reports  

> 💡 *No macros or add-ins required — fully compatible with Excel for Web.*
